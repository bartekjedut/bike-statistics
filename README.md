# General information
Bike Statistics is application dedicated for storing history of bike trips. It allows to store, edit and look over your tours in easy way. Additional feature is viewing summative statistics for selected period.

# Technologies used
*Java FX 11.0.1
*SQLiteJDBC 3.28.0

# How to launch
In order to launch this application, run Main function in Class Main.java.
